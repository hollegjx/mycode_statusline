## Usage

After installation, ccline is automatically configured for Claude Code at `~/.claude/uucode/uucode`.

You can also use it directly:

```bash
uucode --help
uucode --version
```

## For Users in China

Use npm mirror for faster installation:

```bash
npm install -g @gjx1/uucode --registry https://registry.npmmirror.com
```
